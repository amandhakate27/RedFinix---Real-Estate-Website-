# RedFinix 

A modern real estate web application for browsing luxury property listings, viewing detailed property info, and scheduling viewings — built with HTML, Tailwind CSS v4, and vanilla JavaScript.

## Features

- Browse luxury property listings
- View detailed property information
- Schedule property viewings
- User registration and login
- Responsive design with mobile navigation

## Tech Stack

- HTML
- Tailwind CSS v4
- JavaScript

## Pages

- **Home** – Hero section with featured property
- **Properties** – Property listings
- **Show Property** – Detailed property view
- **Schedule** – Schedule a property viewing
- **Schedule Confirmation** – Booking confirmation
- **Login / Register** – User authentication
- **About** – About RedFinix
- **Contact** – Contact form
